# Trackside Café V2

Polished single-page concept for Trackside Café in Flushing, Michigan.

## Direction

- Warm railroad-postcard feel
- Sepia/golden-hour palette without going muddy
- Existing Trackside photos and menu assets reused
- Mobile-first layout
- Stronger hero, menu, pies, gallery, and visit sections
- Subtle fades instead of noisy animation

## Current Build

- `index.html` is self-contained with inline CSS and minimal JavaScript.
- Existing images are referenced from `assets/`.
- Menu pages open directly from the homepage.
- Contact, hours, phone, Facebook, and map links are included.
